NLP based Emotion Detector – Mini Project
This project detects human emotions in real-time using a webcam.
It uses Python, OpenCV, and a trained Deep Learning model to classify emotions such as Happy, Sad, Angry, Neutral, etc.

-Features
1.Real-time face & emotion detection
2.Uses OpenCV for face detection
3.Deep learning model for emotion classification
4.Simple and user-friendly
5.Easy to run on any system
